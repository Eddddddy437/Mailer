這是我的第一個 **Python** 專案，用來自動化處理 Excel 寄信。
### 🚀 如何使用：
1. **安裝環境**：確保電腦已安裝 `pandas` 等套件。
2. **執行程式**：執行檔案清單中的 `mailer.py`。
3. **選擇excel檔，並將mail資訊先維護在mail裡**: 收件者信箱、副本收件者、主旨、內容、附件檔案的完整路徑。
### 💡 維護時的 4 個小提醒

* **📂 附件路徑要寫對**
    * 程式會自動檢查檔案是否存在。如果路徑錯誤（例如檔案在 E 槽卻寫 D 槽），該封信件會被跳過以確保安全。
    * **多個附件：** 請使用英文分號 `;` 隔開。
* **↩️ 換行符號**
    * 在 Excel 單格內按 `Alt + Enter` 產生的換行，程式會自動轉換為 HTML 的 `<br>`，寄出的信件將完美保留排版。
* **✒️ 簽名檔處理**
    * 程式會自動抓取你 Outlook 預設的簽名檔。
    * **註：** 執行時 Outlook 視窗會閃爍一下，這是程式在抓取簽名資訊，屬於正常現象。
* **⚠️ 欄位名稱**
    * 請確保 Excel 的標題與程式要求一致（注意大小寫），否則程式會找不到資料喔！
