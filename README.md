這是我的第一個 **Python** 專案，用來自動化處理 Excel 寄信。
### 🚀 如何使用：
1. **安裝環境**：確保電腦已安裝 `pandas` 等套件。
2. **執行程式**：執行檔案清單中的 `mailer.py`。
3. **選擇excel檔，並將mail資訊先維護在mail裡**: 收件者信箱、副本收件者、主旨、內容、附件檔案的完整路徑。
4. 💡 維護時的 4 個小提醒
附件路徑要寫對：
程式會檢查檔案是否存在。如果路徑寫錯（例如 D:\test.txt 但檔案其實在 E:\），程式會為了保險起見直接跳過該行，不寄出郵件。
如果有多個附件，請用分號 ; 隔開，例如：C:\file1.pdf; C:\file2.jpg。
換行符號：
你在 Excel 單格內按 Alt + Enter 產生的換行，程式碼已經幫你寫了轉換邏輯（replace("\n", "<br>")），所以寄出的信件會保有換行效果，這點不用擔心！
簽名檔處理：
這段程式有個很聰明的設計：它會先 mail.Display() 打開視窗抓取你 Outlook 預設的簽名檔，再把 Excel 的內容塞進去。
注意： 執行時你會看到 Outlook 視窗閃一下，這是正常的。
